# 🌍 AI Disaster Damage Assessment System

An AI-powered system that analyzes **satellite images before and after disasters** to detect damaged buildings and affected areas.  
This project helps **governments, NGOs, and rescue teams** quickly assess disaster impact and respond faster.

---

## 🚀 Project Overview

Natural disasters like floods, earthquakes, and hurricanes cause massive destruction.  
Traditional damage assessment is **slow and manual**.

👉 This project uses **Artificial Intelligence + Satellite Imagery** to:

- Detect damaged buildings automatically  
- Classify damage severity  
- Visualize affected areas on a map  
- Provide analytics for decision-making  

---


---

## ✨ Features

- 📸 Upload satellite images (before & after disaster)
- 🤖 AI-based damage detection (Deep Learning)
- 🏢 Building damage classification:
  - No Damage
  - Minor Damage
  - Major Damage
  - Destroyed
- 🗺 Interactive map visualization (Leaflet.js)
- 📊 Analytics dashboard (Chart.js)
- ⚡ Fast and user-friendly UI
